### Configuration Requirements:
| Key | Description |
| :--- | :--- |
| `SESSION_ID` | The ID generated in Step 2 |
| `OWNER_NUMBER` | Your number in international format (e.g. 254...) |
| `HEROKU_API_KEY` | API key from Heroku Account Settings |
| `HEROKU_APP_NAME` | The name of the app you are creating |
